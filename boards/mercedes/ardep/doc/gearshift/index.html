Gearshift¶

  • Pin-mapping

Starting with rev0.2 the mainboard comes with a “Gearshift“ config header. It allows the user to select different software configurations via a hardware jumper.

The µC pins have to be configured with an internal pullup. Placing a jumper will then pull the corresponding pin low. Using purely software-defined pullups is possible because the state will only be evaluated after booting. Therefore an undefined state, before initializing the IOs can be tolerated.
