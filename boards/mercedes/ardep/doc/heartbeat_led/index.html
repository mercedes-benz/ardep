Heartbeat LED¶

  • Pin - mapping

The mainboard is equipped with a red/green status LED which can be used to indicate different operating states.

Pin - mapping¶

The status LED is configured as active-low. It is connected to the microcontroller as indicated in the following table.

µC Pin

LED Color

PC3

Red

PA3

Green
